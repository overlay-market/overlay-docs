# Perps Listing Application

This guide explains how token projects can apply to list their token as a perpetual (perp) market on Overlay.

## Purpose

The listing application is a structured form that allows project teams to propose their token as a new perp market on Overlay. The information collected helps the risk and protocol teams evaluate whether the token is suitable for a derivatives market.

## Application Form Requirements

A complete application should include the following information:

1. **Project Basics**

   - Token Name
   - Token Symbol
   - Chain
   - Contract Address

2. **Community**

   - X (Twitter)
   - Telegram
   - Discord

3. **Market Information**

   - CoinGecko / CoinMarketCap Link

4. **Operational Contact**

   - Name
   - Email
   - Telegram Handle

5. **Additional Information**

   - Website
   - Docs / Whitepaper
   - Short Project Description

## Review Process

1. **Risk Evaluation**

   Risk contributors evaluate key characteristics of the token including volatility, liquidity depth, oracle reliability, and other risk parameters required to safely support a perp market.

2. **Launch**

   If the market passes risk evaluation and the listing is approved, the perp market will be deployed after final coordination and agreement with the project team.

Submit your token listing request through the official Overlay listing application form.
